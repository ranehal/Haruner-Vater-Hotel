## Deployment Guide (How to get this Live)
**Visual:** GitHub Repo page.

"Want to run this yourself? It's easy."

1. **Clone the Repo:** `git clone <repo-url>`
2. **Backend:** Navigate to `backend/`, update `application.properties` with your MySQL creds, and run `./mvnw spring-boot:run`.
3. **Frontend:** Navigate to `frontend/`, run `npm install` and `npm run dev`.
4. **Live Demo:** For a live demo, I recommend deploying the backend to **Render** or **Railway** (they have free tiers for Java/Docker) and the frontend to **Vercel** or **Netlify**.
